Programming microcontrollers on Linux

Programming Arduino boards

  1. Open Arduino IDE
  2. Select correct board, plug in your board and select correct port: /dev/ACM0 or /dev/ttyUSB0
  3. Write in terminal: sudo chmod a+rw /dev/ttyUSB0
  4. Done.
